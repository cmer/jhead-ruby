Description

This file is a wrapper for the jhead command line tool, written by Matthias Wandel. www.sentex.ca/~mwandel/jhead/

Example: 

  photo = Jhead.new("photo.jpg")
  photo.date_time # => Fri Aug 27 15:53:53 1000 2010

  Jhead.new("photo.jpg") do |p|
    p.date_time = Time.now
  end
